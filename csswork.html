article on the difference between inline,enternal and external in css

inline css

Inline CSS allows you to apply a unique style to one HTML element at a time. one can assign CSS to a specific HTML

element by using the style attribute with any CSS properties defined within it.

enternal css

Internal CSS is used to style a specific HTML page. You can write Internal CSS using the "style tag"

It allows you to use selectors. eg:- id, class, tag name, etc.

external css

External CSS is used to change the look of an entire website by changing just one file.You can write External CSS in a .css file.

CSS external stylesheet means that you upload all styling properties and values to a separate .css file. Then, you link the document to your website.
